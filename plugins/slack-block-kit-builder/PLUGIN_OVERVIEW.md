# Plugin Overview

## Components

### Skill: slack-block-kit

Comprehensive reference for Slack Block Kit including:
- All block types and their fields
- Interactive elements and composition objects
- slack-block-builder library patterns
- Best practices and troubleshooting

### Command: /block-kit

Guided UI building for:
- `message` - Channel notifications, bot responses
- `modal` - Forms, confirmations, workflows
- `home` - App dashboards

### Agent: block-kit-expert

Specialized guidance for:
- Architecture decisions
- Layout optimization
- Multi-modal workflows
- Interaction handling

## Surface Support

| Surface | Max Blocks | Primary Use |
|---------|------------|-------------|
| Messages | 50 | Notifications, responses |
| Modals | 100 | Forms, confirmations |
| Home tabs | 100 | Dashboards |

## Technology

- Slack Block Kit (JSON)
- slack-block-builder (JavaScript/TypeScript)
- mrkdwn formatting
